CSG Engine Licensing

CSG Engine is an in-house game engine used by CutawayShot Games to develop games. It is written in Python and C and we offer licensing to other developers. Features of CSG Engine includes but are not limited to:

1 - Simple and easy to use editor

2 - Supporting 2D,2.5D and 3D

3 - Support for over 11 platforms

4 - Easy to expand Python and C source code

5 - Suitable for all types of games

6 - Lightweight

7 - Produces small executable files anywhere from 5 ot 20 megabytes depending on the game

To purchase a license, please contact us at "csgmicrosoftpublishing@gmail.com"
